# Anwesha Sahu Portfolio

A modern, vibrant personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features
- Responsive one-page portfolio experience
- Animated hero section with rotating tagline
- About, education, skills, projects, certifications, competencies, and contact sections
- Dark mode with light mode toggle
- Smooth scrolling and motion-based transitions

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Run Locally
```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## Project Structure
- `src/App.jsx` - Main page sections and content
- `src/index.css` - Global styling and Tailwind setup
- `public/` - Static assets such as the resume PDF
